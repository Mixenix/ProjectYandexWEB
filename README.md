# Портфолио на Flask

Личный сайт-портфолио с блогом, галереей проектов и формой обратной связи.

## Возможности

- Блог с CRUD-операциями (создание, редактирование, удаление постов)
- Страница проектов с разбивкой по категориям (дизайн, Python-разработка)
- Регистрация и аутентификация пользователей (Flask-Login, Werkzeug)
- Контактная форма с отправкой на почту
- Адаптивная верстка (Bootstrap)

## Стек

- **Python 3.8+**, **Flask**
- **SQLAlchemy** — ORM, модели блога и пользователей
- **Alembic** — миграции базы данных
- **Flask-WTF** + **WTForms** — формы с валидацией
- **Flask-Bootstrap** — стили
- **SQLite** — база данных

## Структура

```
app/
  __init__.py       — инициализация, роуты, конфигурация
  models/           — модели SQLAlchemy (Blogs, User, Jobs)
  templates/        — Jinja2-шаблоны (base, index, blog, projects, contacts, edit)
  static/           — CSS, изображения
migrations/         — конфигурация Alembic
wsgi.py             — точка входа
```

## Запуск

Установите зависимости через PyCharm либо вручную. Требуется Python 3.8+.

```bash
python wsgi.py
```

Сайт будет доступен на `http://127.0.0.1:80/`. Для доступа извне можно использовать ngrok.
